Broken Background (2019)

Glitch Art
3900x2700 px

The original file of this piece was a photoshop file I designed as the background for my laptop. I then took that file and converted it to a sound file to put into the audio editing software called Audacity. After changing the audio file in Audacity, I exported it back into a photo and that was the result. I liked the process and results od this process because it had lots of freedom. Art can sometimes be stressful because of our perfectionism. This allowed me to "mess up" the file but it ended up being something visually appealing.
